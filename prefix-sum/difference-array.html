Difference Array

Upcoming · Medium

A difference array is the inverse operation of a prefix sum, designed for fast range update operations.

If you need to perform many range update operations (like "add 5 to all elements from index 2 to 7"), you record only the start and end of each update in a difference array, then compute the prefix sum to get the final result. This flips the use case: instead of fast queries, you get fast updates.
