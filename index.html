Topic 01 · Accumulators & Offsets

Prefix Sum Pattern

In Progress

Prefix sums are an essential tool for O(1) query-based subarray computations. By storing cumulative values, we can answer range sum queries in constant time.

Topic 02 · Sliding & Converging Bounds

Two Pointers Pattern

Upcoming

Use converging pointers scanning from opposite sides or same-direction runners to solve search/sum constraints in linear time.

Topic 03 · Subarray Bounds

Sliding Window Pattern

Upcoming

Optimize nested O(N²) loops into a linear O(N) sweep by maintaining dynamic window boundaries.

Topic 04 · Cycle Detection

Fast & Slow Pointers

Upcoming

Use two pointers moving at different speeds to detect cycles, find midpoints, and solve linked list problems elegantly.

Topic 05 · Pointer Rewiring

Linked List In-Place Reversal

Upcoming

Reverse linked lists or sub-sections in-place by carefully rewiring next pointers without extra space.

Topic 06 · Ordered Elimination

Monotonic Stack Pattern

Upcoming

Maintain a stack of increasing or decreasing elements to efficiently find next greater/smaller elements in O(N).

Topic 07 · Heaps & Selection

Top K Elements Pattern

Upcoming

Use heaps or quickselect to efficiently find the k largest, smallest, or most frequent elements.

Topic 08 · Merge & Intersect

Overlapping Intervals Pattern

Upcoming

Sort intervals by start time and use greedy merging or sweepline techniques to handle overlaps.

Topic 09 · Search Space Reduction

Modified Binary Search Pattern

Upcoming

Apply binary search to rotated arrays, answer spaces, and non-trivial search domains beyond sorted arrays.

Topic 10 · Tree Walking

Binary Tree Traversal

Upcoming

Master inorder, preorder, postorder, and level-order traversals - the building blocks of all tree problems.

Topic 11 · Deep Exploration

Depth First Search (DFS)

Upcoming

Explore as deep as possible before backtracking - works on trees, graphs, and implicit state spaces.

Topic 12 · Level-by-Level

Breadth First Search (BFS)

Upcoming

Explore neighbors first using a queue - ideal for shortest path, level-order, and multi-source problems.

Topic 13 · Grid Navigation

Matrix Traversal Pattern

Upcoming

Apply DFS/BFS to 2D grids for flood fill, island counting, shortest paths, and connected component problems.

Topic 14 · Exhaustive Search

Backtracking

Upcoming

Systematically explore all possibilities by building candidates and pruning invalid paths early.

Topic 15 · Optimal Substructure

Dynamic Programming (DP)

Upcoming

Break problems into overlapping subproblems and cache results - the most powerful pattern for optimization.
